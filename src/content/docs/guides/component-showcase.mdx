## [File Trees](https://starlight.astro.build/components/file-tree/)

import { FileTree } from '@astrojs/starlight/components';

File trees can include highlights, placeholders, and comments

<FileTree>
- data/
  - source/ any source data we need to directly track
  - derived/ derived data we create
- scripts/
  - _deps.R include baseline dependencies for renv
- **renv.lock** highlight renv
- ...
</FileTree>

## [Steps](https://starlight.astro.build/components/steps/)

import { Steps } from '@astrojs/starlight/components';

<Steps>

1. restore the project to install the baseline packages:

   ```r
   renv::restore()
   ```

2. install any additional packages you want

   ```r
   install.packages("ggplot2")
   ```

</Steps>


<Aside type="tip">
You need both indentation and an extra space for any code/other text below the ordered list
to keep the parser happy. Just an indent will not work.
</Aside>
